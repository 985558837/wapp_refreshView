# wapp_refreshView
一个在微信小程序中使用的上下拉刷新组件

参数：

|参数名|类型|说明|
|:-----  |:-----|-----                           |
| custom-class | 自定义样式class-name | 自定义样式 |
| refresh |boolean   |是否需要刷新|
| loadMore |boolean   |蓝牙  0-无，1-有|
| length |number   |数据长度|
| emptyText |string   |数据为空时的提示|
| noMore |boolean   |是否没有更多了|
| noMoreText |string   |没有更多数据时的提示|
